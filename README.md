# 黄金价格监控桌面小部件

一个Windows桌面小部件，实时显示黄金价格和分时走势图，支持价格阈值提醒功能。

## 功能特点

- **实时价格显示**: 显示当前现货黄金价格（美元/盎司）
- **分时走势图**: 显示最近2小时的价格走势图
- **价格阈值提醒**: 设置高阈值和低阈值，价格达到阈值时发出系统声音提醒
- **桌面小部件**: 无边框、半透明窗口，始终置顶显示
- **可拖动窗口**: 鼠标拖动窗口到任意位置
- **右键菜单**: 提供设置、刷新、退出等功能
- **自动更新**: 定时自动获取最新价格（默认5分钟）
- **配置持久化**: 自动保存窗口位置和设置

## 系统要求

- **操作系统**: Windows 10/11（理论上支持Windows 7+）
- **Python环境**: 不需要Python环境（已打包为独立可执行文件）
- **网络连接**: 需要互联网连接获取黄金价格数据

## 安装和使用

### 方法一：使用打包版（推荐）

1. 下载 `GoldPriceWidget.exe` 文件
2. 双击运行 `GoldPriceWidget.exe`
3. 程序首次运行会在同目录下创建 `config` 文件夹和配置文件

### 方法二：从源代码运行（开发人员）

1. 确保已安装 Python 3.8+
2. 安装依赖：`pip install -r requirements.txt`
3. 运行程序：`python src/main.py`

## 配置说明

### 配置文件位置
- 打包版：`GoldPriceWidget.exe` 所在目录下的 `config/config.json`
- 开发版：项目根目录下的 `config/config.json`

### 配置文件示例
```json
{
  "high_threshold": 4300.0,
  "low_threshold": 4200.0,
  "window_position": {
    "x": 100,
    "y": 100
  },
  "window_size": [300, 180],
  "update_interval": 300,
  "notifications_enabled": true
}
```

### 配置项说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `high_threshold` | 高阈值（美元/盎司），价格高于此值触发提醒 | 4300.0 |
| `low_threshold` | 低阈值（美元/盎司），价格低于此值触发提醒 | 4200.0 |
| `window_position` | 窗口位置（自动保存） | {"x": 100, "y": 100} |
| `window_size` | 窗口大小（宽×高） | [300, 180] |
| `update_interval` | 价格更新间隔（秒，最小30秒） | 300 |
| `notifications_enabled` | 是否启用提醒 | true |

## 使用指南

### 启动程序
1. 双击 `GoldPriceWidget.exe` 启动程序
2. 程序启动后会在桌面显示一个半透明的小窗口

### 窗口操作
- **拖动窗口**: 鼠标左键按住窗口任意位置拖动
- **右键菜单**: 在窗口上点击右键显示菜单
  - **设置**: 打开设置对话框
  - **刷新**: 手动刷新价格数据
  - **退出**: 退出程序

### 设置价格阈值
1. 在窗口上右键点击，选择"设置"
2. 在设置对话框中输入高阈值和低阈值
3. 点击"确定"保存设置

### 价格提醒
- 当价格达到设定的高阈值时，会播放系统提示音
- 当价格达到设定的低阈值时，会播放系统提示音
- 为避免频繁提醒，同一类型的提醒有60秒冷却时间

## 数据来源

- **价格数据**: 来自金价API (`api.jijinhao.com`)
- **更新频率**: API数据每15秒更新一次，程序默认5分钟获取一次
- **价格单位**: 美元/盎司

## 常见问题

### Q1: 程序启动后没有显示价格？
- 检查网络连接是否正常
- 确认防火墙没有阻止程序访问网络
- 等待程序重试（程序会自动重试3次）

### Q2: 价格提醒没有声音？
- 检查系统声音是否正常
- 确认设置中"启用价格阈值提醒"已勾选
- 检查提醒冷却时间（同一类型提醒60秒内不重复）

### Q3: 如何修改更新频率？
- 右键点击窗口，选择"设置"
- 在"更新频率"中输入秒数（最小30秒）
- 点击"确定"保存

### Q4: 程序无法启动？
- 确认操作系统为Windows
- 尝试以管理员身份运行
- 检查杀毒软件是否误报（程序使用PyInstaller打包，可能被误报）

### Q5: 如何彻底卸载程序？
1. 删除 `GoldPriceWidget.exe` 文件
2. 删除同目录下的 `config` 文件夹
3. 删除同目录下的 `logs` 文件夹（如果存在）

## 日志文件

- `logs/alerts.log`: 提醒记录
- 控制台输出也会显示运行状态和错误信息
